# Good Cup Coffee Co. — Roastery (Phase 1)

An interactive, stylized 3D walkthrough of the Phase 1 single-room roastery (≈ 8 × 3.3 m), built with [Three.js](https://threejs.org/). Everything is in a single self-contained `index.html`.

## View it live

👉 **https://xandro-bit.github.io/goodcup-roastery/**

Drag to orbit · scroll to zoom · use the **Entrance / Far end / Dollhouse** buttons to jump between viewpoints. (Needs a modern browser with WebGL and an internet connection — Three.js loads from a CDN.)

## What's in the room

Laid out from the floor plan:

- **Right (exterior) wall** — 1 kg sample roaster, a steel logging table with laptops, and a black 6 kg drum roaster + destoner, each with an insulated exhaust duct rising to the roof.
- **Left wall** — packaging machines (weigh-filler + sealer), a stainless packaging bench, and steel shelving with crates of green coffee at the far end.
- **Entrance end** — stainless cupping / QC table with degassing bins on the lower shelf, and the tall stainless vertical blender.
- Matte white walls, polished grey microcement floor, daylight windows on the exterior wall, wall-mounted AC units, and a warm-wood accent at the entrance.

## Note

This is a **stylized** interactive scene (clean low-poly), not a photorealistic render. For photorealism, run the render brief through a dedicated renderer.
